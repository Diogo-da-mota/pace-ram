# PACE RAM - Sua Plataforma de Corridas

## Project info

**URL**: https://www.paceram.com.br

## Sobre o Projeto

PACE RAM é uma plataforma profissional para fotografia de corridas de rua e eventos esportivos.

## Tecnologias Utilizadas

Este projeto foi construído com:

- Vite
- TypeScript
- React
- shadcn-ui
- Tailwind CSS

## Como Rodar Localmente

Certifique-se de ter Node.js & npm instalados.

```sh
# Passo 1: Clone o repositório
git clone <YOUR_GIT_URL>

# Passo 2: Entre no diretório
cd pace-ram

# Passo 3: Instale as dependências
npm i

# Passo 4: Inicie o servidor de desenvolvimento
npm run dev
```
